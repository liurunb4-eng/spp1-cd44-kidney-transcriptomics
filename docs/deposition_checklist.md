# Deposition Checklist

Before public release or journal submission, check the following items:

- Remove local absolute paths from scripts where practical, or document required path edits.
- Confirm that no raw controlled-access, private, or unpublished data are included.
- Confirm that large public matrices are excluded from Git and downloaded from original repositories.
- Add a repository URL to the manuscript Code Availability statement.
- Add a DOI after Zenodo archival if required by the journal.
- Re-run the final scripts or document which outputs are provided as manuscript supplementary material.
