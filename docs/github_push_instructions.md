# GitHub Push Instructions

Repository:

```text
https://github.com/liurunb4-eng/spp1-cd44-kidney-transcriptomics
```

This local deposition package has been pushed to GitHub. To reconnect a fresh local copy to the same remote, run:

```bash
git remote add origin https://github.com/liurunb4-eng/spp1-cd44-kidney-transcriptomics.git
git push -u origin main
```

Use the repository URL in the manuscript Code Availability statement:

```text
Analysis scripts, module gene sets and signature definitions are available at https://github.com/liurunb4-eng/spp1-cd44-kidney-transcriptomics.
```

For journal submission, a DOI can be created by linking the GitHub repository to Zenodo and archiving a release.
