# GitHub Push Instructions

Suggested repository name:

```text
spp1-cd44-kidney-transcriptomics
```

Create an empty GitHub repository with this name, without adding a README, license, or `.gitignore` on GitHub. Then run the following commands from this local repository:

```bash
git remote add origin https://github.com/<your-username>/spp1-cd44-kidney-transcriptomics.git
git push -u origin main
```

After the push succeeds, use the repository URL in the manuscript Code Availability statement:

```text
Analysis scripts, module gene sets and signature definitions are available at https://github.com/<your-username>/spp1-cd44-kidney-transcriptomics.
```

For journal submission, a DOI can be created by linking the GitHub repository to Zenodo and archiving a release.
